# Anniversary Surprise

Website ini dibuat sebagai hadiah anniversary bernuansa romantis, elegan, dan modern.

## Struktur

- `index.html`
- `css/style.css`
- `js/script.js`
- `assets/images/photo1.jpg`
- `assets/images/photo2.jpg`
- `assets/images/photo3.jpg`
- `assets/music/our-song.mp3`
- `assets/icons/favicon.ico`

## Fitur

- Loading screen dengan fade in / fade out
- Hero fullscreen dengan tombol scroll
- Hal yang aku syukuri dan interaktif flip card
- Timeline mimpi, surat digital, relationship counter
- Memory gallery dengan auto slide dan swipe mobile
- Audio player lokal dengan tombol surprise
- Confetti, floating hearts, popup romantis
- Progress bar scroll dan back-to-top
- Responsive tablet / mobile / desktop
- SEO dasar dan Open Graph untuk preview

## GitHub Pages

Kamu dapat langsung meng-upload repository ke GitHub Pages tanpa konfigurasi tambahan.
Cukup taruh file di branch `main` atau `gh-pages`, kemudian aktifkan GitHub Pages pada repository.

## Tanpa Build

Website ini 100% statis, tanpa Node.js, npm, backend, atau framework.
